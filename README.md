# Django Blog

## Overview

A full-stack blog application built with Django that allows users to create, edit, and manage blog posts through a clean and responsive interface.

## Features

- User authentication
- Create, edit, and delete blog posts
- Responsive web interface
- Django admin panel
- Database integration

## Technologies

- Python
- Django
- HTML
- CSS
- SQLite
- Git
- GitHub

## What I Learned

This project strengthened my understanding of Django fundamentals, including routing, templates, models, authentication, CRUD operations, and database management.

## Installation

```bash
git clone https://github.com/Nally591/django-blog.git

cd django-blog

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
